# 101 Spec Kit

## How to Use

### Step-by-step

1. **Plan business term**

    ```# AI chat command
    /plan business term
    ```

2. **Specify technical term**

    ```# AI chat command
    /specify technical term
    ```

3. **List tasks**

    ```# AI chat command
    /tasks
    ```

4. **Implement a spec**

    ```# AI chat command
    implement specs/xxx-setup-golang-project
    ```

### Example

Suppose you want to set up a new Golang project:

```# AI chat command
/plan business term
/specify technical term
/tasks
implement specs/xxx-setup-golang-project
```
